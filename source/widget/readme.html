Journey Planner Widget - Readme

Download the Journey Planner Widget

To use the Journey Planner Widget in your application:

  1. Copy javascripts/jp_widget.js into your application and source it in your html:
    <script src="javascripts/jp_widget.js" type="text/javascript"></script>
  2. Download the jquery lib and source it in your html:
    <script src="javascripts/jquery-min.js" type="text/javascript"></script>
  3. Get a free licence key here for your website (i.e. yoursite.co.nz) and source the addressfinder libraries in your html (replace LICENSE_KEY with your key):
    <script src="http://api.addressfinder.co.nz/af?key=LICENSE_KEY&v=1" type="text/javascript"></script>
  4. Create an empty div with an id of 'jp_widget' where you want the widget to be placed:
    <div id="jp_widget"></div>
  5. Create the Journey Planner Widget on the jp_widget div:
    $(document).ready(function() {
  $('#jp_widget').journeyPlannerWidget();
});
  6. If you want to set a default destination, pass the following parameters to journeyPlannerWidget:
    a1: the address
x1: x co-ordinate
y1: y co-ordinate
    The exact parameters can be found using AddressFinder through the widget or the Journey Planner website. e.g: 
    $('#jp_widget').journeyPlannerWidget({a1: '40 Bowen Street, Pipitea, Wellington 6011', x1: '174.776742217208', y1: '-41.2787635177392'});
